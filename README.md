# DevRoast

Plataforma para avaliar e ranquear código de forma brutalmente honesta.

## Funcionalidades

- **Envio de Código**: Cole seu código e receba uma avaliação
- **Modo Roast**: Avaliação sem restrições ou cheia de críticas
- **Leaderboard**: Veja os códigos mais criticados da comunidade
- **Preview em Tempo Real**: Syntax highlighting com Shiki

## Stack

- Next.js 16 (App Router)
- Tailwind CSS + tailwind-variants
- Radix UI
- Shiki (syntax highlighting)

## Começando

```bash
npm install
npm run dev
```

Acesse http://localhost:3000
